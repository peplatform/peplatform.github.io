Building the Shared Source Common Langauge Infustructure to XP/2003 Production

First read the readme first in the introductory instructions.

Build the SSCLI with Visual Studio 2005 x86 Command Prompt.

Install the Windows Xp/2003 DDK and launch a xp or 2003 x86 free environment.

build -cz

Copy from the NT Directory the Public Directory and your Developer Directory to the SSCLI root.

Set the INCLUDE dircetories and the LIB Directory in the Public Internal Directory.In a batch file in the Developer Directory named setenv.bat

Copy the ZTOOLS Project to SDKTOOLS and build it with the Build Utlity.

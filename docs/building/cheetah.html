    To Build GNUMake

    To build with nmake on Windows NT, Windows 95, or Windows 98:

    1. Make sure cl.exe is in your %Path%. Example:

    set Path=%Path%;c:/msdev/bin

    2. Make sure %include% is set to msvc include directory. Example:

    set include=c:/msdev/include

    3. Make sure %lib% is set to msvc lib directory. Example:

    set lib=c:/msdev/lib
    4. nmake /f NMakefile

    A short cut to steps 1, 2, and 3 is to run VCVARS32.bat before invoking nmake. For example:

    c:
    cd \msdev\bin\VCVARS32.bat
    cd \path\to\make-3.79
    nmake /f NMakefile

    You might want to start building or porting gnumake, CC, SED, Bison, and M4 first. Use the NT Batch files in the Cheetah projects to configure and Interix.

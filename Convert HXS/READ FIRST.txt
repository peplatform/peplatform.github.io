UPDATED January, 2012 
______________________________________

There are 3 separate utilties here:


1) Folder: Multiple HXS to CHM conversion

  ConvertToCHM2.vbs -- Drop a folder full of HXS (HTML Help 2) files
onto this script and get a folder full of equivalent CHM files. (There are
several improvements to this updated converter. See the help
file for more specific details and options.)
  

2) Folder: HTA

HXSConv.hta -- This is a fullscale GUI version of
the converter. It allows one to select settings and customization, then
convert HXS files, without needing to edit script.


3) Folder: Web Help creator

Scripts to create a master help viewer, in the form of an HTA,
for any number of HXS files -- similar to the MSDN help system.
See the directions file in that folder for full instructions.


------------------- REQUIREMENTS: ------------------------------

>    You need to install 7-Zip for all scripts.
      
>    MS HTML Help Workshop is required for ConvertToCHM2.vbs and for the HTA.

>    With ConvertToCHM2.vbs, the path of HTML Help Workshop must 
        be hardcoded before use.

>    All files must be kept together. The HTA is self-contained within the
      HTA folder. That folder can be copied elsewhere, but the files inside
      ned to be kept together.

------------------- CUSTOM OPTIONS: --------------------------------------

  In addition to required settings, each script also
has optional settings. Those can be set -- and are explained -- near the
top of each script.

  In the HTA the options can be selected easily via checkboxes and text boxes.
There is no need to do any script editing with the HTA.

  There is also a very handy, automatic option that does not need to
be selected. All CHM files are produced with a Favorites tab on the
left side that provides the option to save links to often-visited topic pages.


  These scripts will produce results fairly quickly, but the problem
of MSHelp links slows things down.  (See the help file for more details.)
Average size HXS files will process in less than 1 minute.

Note about converting multiple HXS files:

*** THERE ARE REQUIRED SETTINGS BEFORE YOU CAN USE THIS SCRIPT ***


To adjust the settings, just open ConvertToCHM2.vbs in a text editor.
Look near the top where it says:
 
'--====================SETTINGS=============================

 You *MUST* enter the path to the CHM compiler and 7-Zip, near the top.

Most of the other settings are not important, but two are worth considering:

CleanScriptMess - Set to true to avoid script errors and prevent
hiding of elements in a page.

CustomStyleBlock -  The basic style formatting that MS uses is simple, with large 
text. Like the HTA version, you can choose to add CSS to each page in the help 
files in order to customize their look. CustomStyleBlock is where you can add CSS. 
Anything you like. It will be added to the header of each page, overriding the 
existing style sheets. Just set it up like it would be in an HTML file.
Example:

 CustomStyleBlock =  "<STYLE>" & vbCrLf & "BODY, TD, DIV, P, SPAN {font-family: verdana, sans-serif; font-size: 12px; line-height: 1.4; color: #000040; background-color: #FEFDFC;}"
  CustomStyleBlock = CustomStyleBlock & vbCrLf & "H1, H2, H3, H4 {font-size: 18px; color: #0033AA; background-color: #FFF0DF; font-family: arial, helvetica, sans-serif;}"
  CustomStyleBlock = CustomStyleBlock & vbCrLf & "PRE {font-size: 12px; color: #660000; line-height: 1.4;}" & vbCrLf & "A:link {color: #0033FF; text-decoration: none;}" & vbCrLf & "A:visited {text-decoration: none;}"
  CustomStyleBlock = CustomStyleBlock & vbCrLf & "A:hover {color: #FFFFFF; background-color: #0066EE; text-decoration: none;}" & vbCrLf & "</STYLE>"
